## Building

- Lexsur is served using Node.js. On MacOS:

`brew install node`

- Dependencies can then be installed using NPM (packaged with Node) or Yarn

`$ npm install`
`$ yarn`

- Dev server with reloading

`$ npm run dev`

- Mocha tests

`$ npm run test`

- Production mode server

`$ npm run start`
